# Home Assistant App: Tronbyt Server

[Tronbyt Server](https://github.com/tronbyt/server) for Home Assistant.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Build Status](https://img.shields.io/github/actions/workflow/status/kaffolder7/ha-app-tronbyt-server/test.yml)

## About

You can use this app to manage your Tronbyt/Tidbyt displays locally. It provides a web interface for configuring and managing your displays, as well as a command-line interface for advanced users.
